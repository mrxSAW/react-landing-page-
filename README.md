# React Landing Page

## 📌 Description

React Landing Page est une application web moderne développée avec React JS et Vite. Elle permet de présenter un produit, un service ou une solution digitale à travers une interface attractive, responsive et interactive.

Le projet a été réalisé dans le cadre d'un exercice de développement Front-End afin de mettre en pratique les concepts fondamentaux de React, les composants réutilisables, la gestion d'état et le responsive design.

---

## 🚀 Technologies utilisées

* React JS 19
* JavaScript ES6+
* HTML5
* CSS3
* Vite
* Git & GitHub
* EmailJS

---

## ✨ Fonctionnalités

### Header / Navbar

* Logo de la marque
* Navigation entre les sections
* Bouton d'appel à l'action
* Menu hamburger responsive

### Hero Section

* Titre principal
* Description du service
* Bouton principal
* Illustration

### Features Section

* Présentation des fonctionnalités
* Cartes réutilisables avec props

### About Section

* Présentation de la solution
* Explication des avantages

### Testimonials Section

* Avis clients
* Retours utilisateurs

### Contact Section

* Formulaire de contact
* Validation des champs
* Vérification de l'adresse email
* Envoi d'email avec EmailJS
* Messages de succès et d'erreur

### Footer

* Liens utiles
* Informations de copyright

### Interactions JavaScript

* Mode sombre / clair
* Bouton retour en haut
* Menu responsive mobile
* Validation du formulaire
* Affichage dynamique des messages

## ⚙️ Installation

Cloner le projet :

```bash
git clone https://github.com/mrxSAW/react-landing-page-.git
```

Accéder au dossier :

```bash
cd react-landing-page-
```

Installer les dépendances :

```bash
npm install
```

Lancer le projet :

```bash
npm run dev
```

Ouvrir ensuite :

```text
http://localhost:5173
```

## 📱 Responsive Design

L'application est adaptée aux :

* Ordinateurs
* Tablettes
* Smartphones

Grâce à l'utilisation des Media Queries CSS.

---

## 🌙 Dark Mode

L'utilisateur peut basculer entre :

* Mode clair
* Mode sombre

Le changement est effectué dynamiquement grâce à React et au Hook `useState`.

---

## 📧 Intégration EmailJS

Le formulaire de contact utilise EmailJS pour envoyer les messages directement depuis le Front-End sans avoir besoin d'un serveur Backend.
